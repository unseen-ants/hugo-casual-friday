Welcome to **Casual Friday**, a warm and inviting Hugo theme designed for personal blogs and portfolios.

## Features

This theme comes packed with thoughtful features:

- **Dark Mode** - Automatic detection with manual toggle
- **Responsive Design** - Looks great on all devices
- **Configurable Colors** - Choose from presets or use custom colours
- **Audio Player** - Optional "Listen to Articles" feature

## Typography

The theme uses a carefully selected font pairing:

- **Headings** use Playfair Display for an elegant, editorial feel
- **Body text** uses Source Serif 4 for excellent readability
- **Code blocks** use JetBrains Mono for clear monospace text

### Code Example

Here's how you might configure the theme:

```toml
[params]
  accentColor = "cerulean"
  backgroundScheme = "warm"
```

## Getting Started

1. Install the theme in your Hugo site
2. Copy the example `hugo.toml` configuration
3. Add your content to `content/posts/`
4. Run `hugo server` to preview
